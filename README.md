# CaseFlow – Smart Complaint Management System

## Overview
CaseFlow is a web-based complaint management system developed using Python Flask.  
It helps users register complaints, track complaint status, and manage complaint records through a simple and user-friendly interface.

## Features
- User complaint registration
- Complaint tracking
- Complaint status updates
- Admin complaint management
- Simple dashboard interface
- Database integration for storing complaint records

## Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite / MySQL (use whichever you used)
- Bootstrap (if used)

## Project Objective
The objective of this project is to simplify complaint handling by providing a centralized platform for registering and managing complaints efficiently.
